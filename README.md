# 🌮 Tacos Esther - Sistema de Punto de Venta (POS)

Sistema integral de gestión para taquerías desarrollado en **Python**. Este software automatiza el ciclo de venta, desde la autenticación del personal hasta la emisión de tickets térmicos profesionales.


## 🚀 Funcionalidades Clave
* **Seguridad:** Módulo de login con validación de credenciales en base de datos.
* **Interfaz Dinámica:** Menú interactivo con soporte para modificadores (ej. "Con Queso").
* **Generación de Tickets:** Motor de PDF configurado para formato térmico de 80mm.
* **Base de Datos:** Persistencia mediante SQLite3 para registro histórico de ventas.
* **Manejo de Errores:** Validación de entradas de pago y prevención de fallos críticos.

## 🛠️ Stack Tecnológico
* **Lenguaje:** Python 3.12
* **GUI:** Tkinter (Interfaz Gráfica)
* **Documentación:** FPDF (Generación de Tickets en PDF)
* **Persistencia:** SQLite3 (Base de Datos Local)
